This applet is designed to allow you to easily and securely access the Grid using GSISSH. By default it should work with any LRZ, PRACE and EGI resources where you have an account using your certificate or a proxy certificate stored in the MyProxy server. For more details click here or read the NGS's FAQ.

Please note, you will require Java Software Development Kit (known as a JDK or SDK) 1.6 or higher installed to launch the applet!
